# Screenshots

This directory contains screenshots for the marketplace listing.

## Required Screenshots

1. `typography-showcase.png` - Typography showcase page showing all variants
2. `login-page.png` - Login page in both light and dark themes

## How to Add Screenshots

1. Take screenshots of the application
2. Save them in this directory with the names above
3. Commit and push to GitHub
4. The marketplace.json will automatically reference them
