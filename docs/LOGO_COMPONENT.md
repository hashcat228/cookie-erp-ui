# Logo Component

Переиспользуемый компонент логотипа компании с поддержкой темной темы и типографики.

## Использование

```tsx
import { Logo } from "@/components/logo"

// Базовое использование
<Logo />

// Без текста (только иконка)
<Logo showText={false} />

// Разные размеры
<Logo size="sm" />  // Маленький (6x6)
<Logo size="md" />  // Средний (8x8) - по умолчанию
<Logo size="lg" />  // Большой (10x10)

// Кастомная ссылка
<Logo to="/dashboard" />
```

## Особенности

- ✅ Использует Typography компонент для текста
- ✅ Адаптируется к темной/светлой теме
- ✅ Hover эффекты на группе
- ✅ Семантический HTML с aria-label
- ✅ Три размера: sm, md, lg
- ✅ Опциональный текст
- ✅ Настраиваемая ссылка

## Лучшие практики

1. **Семантика**: Логотип обернут в `<Link>` с `aria-label` для доступности
2. **Типографика**: Текст использует Typography компонент с правильными вариантами
3. **Hover состояния**: Применяются на уровне группы для плавных переходов
4. **Иконка**: `aria-hidden="true"` на декоративном изображении
5. **Цвета**: Использует семантические токены (`bg-primary`, `text-primary`)
6. **Размеры**: Три предустановленных размера для разных контекстов

## Примеры использования

### В шапке страницы
```tsx
<header className="flex justify-between items-center p-6">
  <Logo />
  <ThemeToggle />
</header>
```

### В сайдбаре
```tsx
<aside className="w-64 p-4">
  <Logo size="sm" />
  {/* Навигация */}
</aside>
```

### Только иконка в компактном режиме
```tsx
<div className="collapsed-sidebar">
  <Logo showText={false} size="sm" />
</div>
```
